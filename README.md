## Featured Projects

- **[Shikkha Sahayok](https://github.com/virtualofficepro/learning-platform-backend)**
  An AI-powered tutoring platform for Bangladeshi primary students (Classes 1–5), aligned with the NCTB curriculum.
  Built with a RAG pipeline on Bengali textbooks, multilingual embeddings, and a child-friendly Flutter interface.
  *(Django · FastAPI · ChromaDB · Groq · Flutter)*

- **[IELTS AI Coach](https://github.com/shaDC24/IELTS-AI-COACH)**
  A full-stack IELTS preparation platform with Writing Coach, Speaking Coach (Whisper), Study Plan Generator, and AI Mentor Chat.
  *(FastAPI · React · Groq · NeonDB · Render · Vercel)*

- **[PCOS Detection](https://github.com/shaDC24/ML_PROJECT_PCOS_DETECTION)**
  ML-based clinical diagnosis system for Polycystic Ovary Syndrome from structured medical data.

- **[E-Clinic Assistant](https://github.com/shaDC24/E-clinic-Assistant---Atmega32-project)**
  An embedded clinic management assistant built on ATmega32 — my favourite kind of problem: constrained, physical, real.
